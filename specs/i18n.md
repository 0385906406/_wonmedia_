# Internationalization (i18n) — WonMedia CMS

## Ngôn ngữ hỗ trợ

| Code | Ngôn ngữ | Flag |
|------|----------|------|
| `vi` | Tiếng Việt | 🇻🇳 (default) |
| `en` | English | 🇺🇸 |
| `ko` | 한국어 | 🇰🇷 |
| `ja` | 日本語 | 🇯🇵 |
| `zh` | 中文 | 🇨🇳 |

## Cấu trúc file

```
dictionaries/
  vi.json          # Bản dịch Tiếng Việt
  en.json          # English
  ko.json          # 한국어
  ja.json          # 日本語
  zh.json          # 中文

app/(client)/[lang]/
  dictionaries.ts  # Server-only loader (import 'server-only')
  language-switcher.tsx  # Client Component dropdown
  layout.tsx       # Navbar, footer (lang aware)
  page.tsx         # Trang chủ
```

## Routing

```
/        → redirect → /vi (hoặc ngôn ngữ từ Accept-Language)
/vi      → Trang chủ Tiếng Việt
/en      → English home
/ko      → Korean home
/ja      → Japanese home
/zh      → Chinese home
```

## Proxy (proxy.ts)

- Đọc `Accept-Language` header qua `@formatjs/intl-localematcher` + `negotiator`
- Nếu path chưa có locale prefix → redirect `/${locale}${path}`
- Skip: `/_next`, `/api`, `/admin`, `/auth`, static files

## Dictionary Schema

```json
{
  "locale": "vi",
  "nav": { "home", "services", "support", "contact", "login" },
  "hero": { "badge", "title", "subtitle", "cta_primary", "cta_secondary" },
  "stats": { "users", "uptime", "speed", "support" },
  "services": { "eyebrow", "title", "subtitle", "fiber", "business", "cloud" },
  "cta": { "badge", "title", "subtitle", "button", "secondary" },
  "footer": { "tagline", "rights" }
}
```

## Thêm ngôn ngữ mới

1. Tạo `dictionaries/{lang}.json` (copy từ `en.json`, dịch giá trị)
2. Thêm vào `locales` trong `proxy.ts`
3. Thêm vào `dictionaries` object trong `app/(client)/[lang]/dictionaries.ts`
4. Thêm label trong `LANG_LABELS` trong `language-switcher.tsx`
5. Thêm vào `generateStaticParams` trong `app/(client)/[lang]/layout.tsx`
